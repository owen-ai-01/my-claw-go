# MyClawGo 推广与扩展页面计划

**日期：** 2026-05-04  
**参考竞品：** myclaw.ai  
**目标：** 在 3 个月内将自然流量提升至 myclaw.ai 的 20% 水平，重点攻占"multi-agent"和"openclaw hosting"两个关键词赛道

---

## 一、竞品现状分析（myclaw.ai）

### 核心数据
- **博客文章：** 79 篇，覆盖安装、配置、工作流等方向
- **定价：** Lite $16/mo → Pro $33/mo → Max $66/mo → Ultra $133+/mo
- **核心页面：** /use-cases（69 个案例）、/compare、/skills、/learn（7 天学习路径）、/tutorial

### MyClawGo 相对优势（myclaw.ai 没有的）
| 功能 | MyClawGo | myclaw.ai |
|---|---|---|
| 多 Agent 群组接龙（relay） | ✅ | ❌ |
| 独立 OpenRouter API Key | ✅ | ❌ |
| 中文界面 + 中文内容 | ✅ | ❌ |
| 无限 Agent 数量（所有套餐） | ✅ 待确认 | 有限制 |
| 专属独立实例 | ✅ | 不明确 |

### myclaw.ai 的弱点
1. **无 multi-agent relay** —— 无法让多个 agent 自动接力完成复杂任务
2. **无中文内容** —— 中文用户搜索结果完全空白
3. **pricing 偏高** —— Pro $33/mo vs MyClawGo Pro $29.90/mo
4. **无 compare 页面主动比较** —— 搜索"myclaw alternative"几乎无结果

## 二、页面建设路线图

按 P0 / P1 / P2 优先级排列。

title: "MyClawGo vs MyClaw.ai — OpenClaw Hosting Comparison 2026"

**目标关键词：** `myclaw alternative`、`best openclaw hosting alternative`、`openclaw managed hosting`

### P1 — 本季度（6 月底前）

**每个场景页面：** 场景描述 + agent 配置示例 + 效果截图 + CTA

#### 4. `/multi-agent` — 核心差异化落地页
**目标关键词：** `multi-agent openclaw`、`openclaw agent groups`、`openclaw relay`

**页面结构：**
- 问题描述：单 agent 的局限性

- 典型用例：研究 + 写作 + 审阅的三段接力
- FAQ：最多几个 agent？如何避免循环？
- CTA：立即创建群组

#### 5. `/learn` — 7 天入门学习路径
**参考：** myclaw.ai 有专门的 /learn 路径，结构化引导新用户

**内容规划：**

- Day 3: 为 Agent 添加工具和系统提示词
- Day 4: 创建 Agent 群组

- Day 6: 连接 Telegram Bot
- Day 7: 高级技巧：自定义模型路由、credit 管理

## 五、定价定位策略

### 当前定价 vs myclaw.ai

| 套餐 | MyClawGo | myclaw.ai |
|---|---|---|
| 入门 | Pro $29.90/mo | Lite $16/mo |
| 中级 | Premium $59.90/mo | Pro $33/mo |
| 高级 | Ultra $199.90/mo | Max $66/mo |

### 分析与建议
myclaw.ai 的入门价格（$16/mo）低于 MyClawGo（$29.90/mo）。在价格对比中需要明确：

**MyClawGo 的价值主张：**
- 包含独立 OpenRouter API Key（myclaw 不含）
- 多 agent 群组功能（myclaw 不含）
- 专属独立实例（非共享）
- 中文支持

**定价页面优化建议：**
1. 在 pricing 页面增加 "vs MyClaw" 对比表
2. 强调"独立 OpenRouter Key = 你自己掌控 API 用量，无 token 上限担忧"
3. 考虑新增 $16-19/mo 的 Starter 套餐以应对 myclaw Lite 的竞争
